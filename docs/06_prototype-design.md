# プロトタイプ設計書　家族向け予定・家計簿アプリ（PlannerwithExpense）

---

## 1. 概要

HTML / CSS / JavaScript を使って、画面イメージと導線を確認するための簡易モック。
本実装（Vue.js + FastAPI）とは別物で、データの保存・API連携は行わない（ページを再読み込みすると初期状態に戻る）。

---

## 2. ファイル構成

```
prototype/
├── login.html         # ログイン画面
├── register.html       # 新規登録画面
├── calendar.html       # カレンダー画面
├── expenses.html        # 家計簿画面
├── css/
│   └── style.css       # 全画面共通スタイル
└── js/
    └── app.js          # モーダル開閉・一覧への仮追加など最小限の動き
```

---

## 3. 使用技術

| 種類 | 技術 |
|------|------|
| マークアップ | HTML5 |
| スタイル | CSS3 |
| 動作 | JavaScript（バニラJS） |
| データ | なし（ページ内の仮データのみ、リロードで消える） |

## 4. 画面ごとの設計

### 4-1. ログイン画面（login.html）

```
┌─────────────────────────────┐
│   PlannerwithExpense          │
├─────────────────────────────┤
│  メールアドレス：[________]    │
│  パスワード　　：[________]    │
│                              │
│        [ ログイン ]           │
│                              │
│  アカウントをお持ちでない方は  │
│  新規登録はこちら              │
└─────────────────────────────┘
```

- 「ログイン」ボタン → calendar.html へ遷移（認証チェックなし）
- 「新規登録はこちら」リンク → register.html へ遷移

### 4-2. 新規登録画面（register.html）

```
┌─────────────────────────────┐
│   新規登録                   │
├─────────────────────────────┤
│  メールアドレス：[________]    │
│  パスワード　　：[________]    │
│                              │
│        [ 登録する ]           │
│                              │
│  すでにアカウントをお持ちの方は │
│  ログインはこちら              │
└─────────────────────────────┘
```

- 「登録する」ボタン → login.html へ遷移（実際の登録処理なし）
- 「ログインはこちら」リンク → login.html へ遷移

### 4-3. カレンダー画面（calendar.html）

```
┌──────────────────────────────────────────┐
│ PlannerwithExpense │カレンダー│家計簿│ログアウト │
├──────────────────────────────────────────┤
│              2026年7月              │
│  日  月  火  水  木  金  土          │
│  ...                                │
│  6   7*  8   9   10  11  12         │
│      [予定A]                        │
│  ...                                │
│                        [ ＋ 予定追加 ] │
└──────────────────────────────────────────┘
```

- ヘッダーのナビ（カレンダー／家計簿）で画面を切り替え
- 日付セル内の予定チップをクリック → 編集モーダル（タイトル・日付・メモ・削除ボタン）
- 「＋ 予定追加」ボタン → 追加モーダル（タイトル・日付・メモ）
- モーダルで「保存」→ 一覧に仮追加してモーダルを閉じる（保存はページ内のみ、リロードで消える）
- 「ログアウト」→ login.html へ遷移

### 4-4. 家計簿画面（expenses.html）

```
┌──────────────────────────────────────────┐
│ PlannerwithExpense │カレンダー│家計簿│ログアウト │
├──────────────────────────────────────────┤
│  日付      カテゴリ  金額    メモ   操作    │
│  07/01     食費     1,200  昼食  [編集][削除]│
│  07/03     日用品   3,000        [編集][削除]│
│                                            │
│                        [ ＋ 支出追加 ] │
└──────────────────────────────────────────┘
```

- 「＋ 支出追加」ボタン → 追加モーダル（金額・日付・カテゴリ・メモ）
- 各行の「編集」→ 編集モーダル（同項目、削除ボタンあり）
- 各行の「削除」→ 確認なしで一覧から仮削除（ページ内のみ）
- 「ログアウト」→ login.html へ遷移

---

## 5. 実装しない機能（プロトタイプ対象外）

| 機能 | 理由 |
|------|------|
| データの保存・DB連携 | 画面イメージ確認が目的のため不要 |
| ユーザー認証（実際のログインチェック） | 本実装フェーズで対応 |
| サーバー・API連携 | 本実装フェーズで対応 |
| バリデーション（入力チェック） | 本実装フェーズで対応 |
| 月・週の切り替えなど実際のカレンダー操作 | 静的な1ヶ月分の表示のみで代用 |
